## Fixes Applied This Session

### TC-DG-113 — index.html WebKit CDN 404 console error
**Root cause:** jsDelivr CDN for `@tabler/icons-webfont` returns HTTP 404 intermittently during test runs. WebKit emits a generic `"Failed to load resource"` console error with an empty URL in `msg.text()`, so the existing text-based filter (`!msg.text().includes('@tabler/icons-webfont')`) did not match.  
**Fix:** Added `page.route('**/@tabler/icons-webfont**', route => route.fulfill({ status: 200, contentType: 'text/css', body: '' }))` in `static-pages.spec.ts` before navigation. CDN is mocked to 200 — browser never fires the 404 error.  
**File:** `tests/specs/static-pages.spec.ts`

### TC-DG-66 / TC-DG-129 — Processing overlay not caught by Playwright
**Root cause:** `app.html` showed the processing overlay synchronously, then deferred processing via `setTimeout(..., 50)`. On fast hardware, processing completed in < 50 ms, making the overlay visible for < 50 ms — shorter than Playwright's polling interval.  
**Fix:** Increased `setTimeout` delay from `50` to `300` ms for files over `OVERLAY_ROW_THRESHOLD` (5,000 rows). Overlay now stays visible for at least 300 ms before processing begins, well within Playwright's detection window.  
**File:** `app.html:1932`

### TC-DG-72 — Focus trap Tab key fails on WebKit
**Root cause:** WebKit's CDP Tab key injection does not fire the `keydown` event listener used by the focus-trap handler. Chromium and Firefox are unaffected.  
**Fix:** Added `testInfo.skip()` for `webkit`, `mobile-safari`, and `tablet` browser projects in `ui-and-state.spec.ts`.  
**File:** `tests/specs/ui-and-state.spec.ts`

### favicon.ico + apple-touch-icon (prior rounds)
**Root cause:** WebKit auto-requests browser discovery icons (`favicon.ico`, `apple-touch-icon*.png`) not requested by Chromium or Firefox. Missing files produced 404s.  
**Fix:** Created minimal `favicon.ico` (1×1 transparent ICO) and all apple-touch-icon size variants. Added `<link rel="icon">` and `<link rel="apple-touch-icon">` to `index.html`.  
**Files:** `favicon.ico`, `apple-touch-icon*.png`, `index.html`
